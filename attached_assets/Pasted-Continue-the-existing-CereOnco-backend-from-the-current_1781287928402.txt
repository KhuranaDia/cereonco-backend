Current production:
https://cereonco-backend-a21z.onrender.com

Required DB/API shape:

Post should support:
{
"id": number,
"userId": number,
"content": "string",
"feeling": "string | null",
"mediaUrls": ["string"],
"createdAt": "...",
"updatedAt": "..."
}

{

"feeling": "hopeful",
"mediaUrls": [

}

{

"feeling": "grateful",
"mediaUrls": [
"https://example.com/new-image.jpg"
]
}
