4. User Search API
==================

Create:

GET /users/search?q=&limit=&offset=

Requirements:

* Auth required.
* Search by name and email.
* Case-insensitive.
* Minimum q length: 2 characters.
* Default limit 20.
* Default offset 0.
* Return only safe user fields:
  id, name, email, role, avatarUrl, profilePhotoUrl, imageUrl if available
* Do not return passwordHash or tokens.

Response:

"success": true,
"message": "Users retrieved successfully",
"data": {
"users": [],

}
